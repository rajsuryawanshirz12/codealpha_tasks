--------------------------------------------------------------------------
DropZone is a fully functional E-Commerce platform dedicated to authentic 
sneakers. This project was built to demonstrate Full Stack development 
skills, integrating a responsive Frontend with a robust Node.js/Express 
Backend and MongoDB Database.

--------------------------------------------------------------------------
üöÄ FEATURES COMPLETED (Internship Checklist)
--------------------------------------------------------------------------
1. Shopping Cart: 
   - Fully functional with persistent state (LocalStorage).
   - Dynamic calculations for subtotals and totals.

2. Product Details: 
   - Individual pages for products (Air Jordan, Yeezy, etc.).
   - "More from the Vault" recommendation engine.

3. Order Processing: 
   - Secure Checkout form.
   - Orders are validated and saved to MongoDB via API.
   - Order Confirmation page.

4. User Authentication:
   - Registration and Login system.
   - User data stored securely in MongoDB.

5. Database:
   - MongoDB used for Users, Products, and Orders.
