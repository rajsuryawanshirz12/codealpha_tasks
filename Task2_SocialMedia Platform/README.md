--------------------------------------------------------------------------
CastCircle is a specialized social media platform focused on audio content 
and podcasting. Unlike generic social networks, this platform allows users 
to share, stream, and discover audio episodes in a modern, glassmorphism-
styled interface. It features a fully functional audio player, user 
profiles, and a dynamic feed.

--------------------------------------------------------------------------
üöÄ FEATURES COMPLETED (Internship Checklist)
--------------------------------------------------------------------------
1. User Authentication:
   - Secure Login & Registration pages with Glassmorphism UI.
   - User sessions managed via LocalStorage.

2. Audio Streaming Engine:
   - Custom-built Sticky Audio Player (plays continuously while browsing).
   - Play/Pause, Progress Bar, and Volume controls.

3. Dynamic Feed & Discovery:
   - Home Feed showing latest podcasts from various categories.
   - Explore Page with trending categories (Tech, Startups, Motivation).
   - Filter system to sort content by genre.

4. User Profiles:
   - Dedicated profile pages for creators (e.g., Lex Fridman, User).
   - Follow/Unfollow system (Visual state).
   - "Library" and "Listen Later" sections for saving content.

5. Interaction System:
   - Like, Comment, and Share visual indicators.
   - Real-time notification simulation.
   - Settings page for profile management.
